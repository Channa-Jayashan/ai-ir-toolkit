# Kali Execute

Runs real Kali tools over local Wi-Fi and returns the output.

## Examples
* "Use kali-execute to run strings on evidence.bin"
* "Run volatility3 with plugin windows.pslist on memory.dump"
* "Calculate sha256sum of suspicious.exe using the Kali bridge"
